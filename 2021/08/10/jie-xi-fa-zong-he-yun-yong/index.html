解析法综合运用

发表于 2021-08-10

作者 / 190331 舒熙皓

解析法 不同于我们常用的利用倒角，全等，平分线等纯几何的方法来解决问题，而是利用建立平面直角坐标系，利用坐标，直线方程，函数等进行代数和几何的综合运用来得出解。事实上，解析法不需要太多的思维，更注重的是计算能力和分析能力。我们来用一些例题探究解析法在初中数学中的运用：

Example 1 / 如图，有正方形 $ABCD$ 和 $BEFG$，边长分别为 4 和 6。连接 $DF$，$H$ 为 $DF$ 中点，连接 $GH$，请求出 $GH$ 的长度。

解：以 $B$ 为原点，$AE,GB$ 所在执行分别为 $x$ 轴和 $y$ 轴，建立平面直角坐标系。由题可得

$D(-4,4),F(6,6),G(0,6)$，由中点公式得 $H(1,5)$，由两点间距离公式得

$|GH|=\sqrt{(0-1)^2+(6-5)^2}=\sqrt{2}$​​​。

从这个例子中，我们可以看到解析法对于此类问题的方便快捷。

Example 2 / 如图，有边长为 4 的正方形 $ABCD$，$E$ 为 $BC$ 中点。作 $DF\perp DE$ 交 $BA$ 延长线于点 $F$。连接 $AC$ 交 $EF,DE$ 于点 $H,G$，求 $GH$ 长度。

解：以 $A$ 为原点，$AB,AD$ 所在执行分别为 $x$ 轴和 $y$ 轴，建立平面直角坐标系。由题可得

$A(0,0),D(0,4),C(4,4),E(4,2)$。由待定系数法可得

$AC:y=x, DE:y=-\dfrac{1}{2}x+4, G(\dfrac{8}{3},\dfrac{8}{3})$。因为 $DF$ 过点 $D$ 且垂直于 $DE$，

$DF:y=2x+4, F(-2,0), EF:y=\dfrac{1}{3}x+\dfrac{2}{3}, H(1,1)$。由两点间距离公式得

$|GH|=\sqrt{2\times (\dfrac{8}{3}-1)^2}=\dfrac{5}{3}\sqrt{2}$​​​​。

从这个例子中，我们发现可以利用直线中两点的坐标，相互垂直的直线之间的斜率关系，一个点的坐标和直线斜率和待定系数法能够快速算出来直线的解析式或者点的坐标。

Example 3（衡阳2019）/ 在平面直角坐标系 $xOy$ 中，二次函数 $y=x^2+bx+c$ 的图象交 $x$ 轴于 $A(-1,0),B(3,0)$，与 $y$ 轴交于点 $N$。以 $AB$ 为边在 $x$ 轴上方作正方形 $ABCD$，点 $P$ 是 $x$ 轴上一动点，连接 $CP$，过点 $P$ 作 $CP$ 垂线与 $y$ 轴交于点 $E$ 。

1 / 求该抛物线的函数解析式。

2 / 当 $P$ 与线段 $OB$（不与 $O,B$ 重合）上运动至何处时，线段 $OE$ 的长有最大值？请求出 $E$ 点的坐标和这个最大值。

3 / 在第四象限的抛物线上任取一点 $M$，连接 $MN,MB$ 。请问：$S_{\triangle MBN}$ 是否存在最大值？若存在，请求出点 $M$ 的坐标。不存在请说明理由。

1 / 由顶点式得 $(x+1)(x-3)=y$，整理得 $y=x^2-2x+3$。

2 / 设 $P(a,0)$，由图得 $C(3,4),D(-1,4), CP:y=\dfrac{-4x+4a}{a-3}$。又因为 $EP\perp PC$，

$k_{EP}=\dfrac{a-3}{4}, EP:y=\dfrac{a-3}{4}(x-a), E(0,\dfrac{a(3-a)}{4})$。

显然，当 $a=3-a=1.5$​ 时，$\dfrac{a(3-a)}{4}$​ 取到最大值，此时 $E(0,\dfrac{9}{16}),OE_\max=\dfrac{9}{16}$​。

3 / 当 $M$​ 作为切点，且切线斜率为 $k_{NB}$​ 时，$S_{\triangle MBN}$​ 取到最大值。利用铅锤法可以算出面积，但我们可以尝试一种更加简单的方法——求导法。

设 $f(x)=x^2-2x+3$​，则 $f’(x)=2x-2$​，$f’(x)$​ 几何意义即为 $f(x)$​ 的切线斜率。由题得

$NB:y=x-3, \therefore 2x-2=1,x=1.5, M(1.5,-3.75)$​​​ 。

从这个例子中，我们可以发现巧妙的设元可以使问题变得更加简单可做，合理利用超纲知识也可以减少不必要的一些计算难度。在解决这种复杂问题的时候，计算错误是一定要注意避免的。

挑战中考

邵阳 2019 / 二次函数 $y=-\dfrac{1}{3}x^2+bx+c$ 的图象过原点和 $(8,0)$。

1 / 求该函数的解析式。

2 / 在 $x$ 轴上方作直线 $y_1=m$，交二次函数于点 $A,B$，过这两点分别作 $x$ 轴的垂线，交 $x$ 轴于点 $D,C$ 。当矩形 $ABCD$ 为正方形时，求 $m$ 的值。

3 / 在 2 的条件下，动点 $P$ 从点 $A$ 出发沿射线 $AB$ 以 $1/s$ 匀速运动，同时动点 $Q$ 以 $1/s$ 从 $A$ 出发沿线段 $AD$ 匀速运动，到达点 $D$ 后原速返回。当 $Q$ 返回点 $A$ 时，$P,Q$ 停止运动。设运动时间为 $t (t>0)$ 秒，过点 $P$ 向 $x$ 轴作垂线，交抛物线于点 $E$，交直线 $AC$ 于点 $F$。请问：以 $A,E,F,Q$ 四点为顶点能否构成平行四边形。若能，请求出 $t$。若不能，请说明理由。

长沙 2018 / 函数 $y=\dfrac{m}{x} (m>1,x>0)$ 的图象过点 $P(m,1),Q(1,m)$ ，直线 $PQ$ 与 $x$ 轴，$y$ 轴交于点 $C,D$，点 $M(x,y)$ 为函数图象上一动点，在坐标轴上有点 $A,B$ 使得四边形 $OAMB$ 为矩形。

1 / 求 $\angle OCD$ 的度数。

2 / 当 $m=3, 1<x<3$ 时，请求出使得 $\triangle OPM\sim \triangle OCP$ 的 $M$ 的坐标。

3 / 当 $m=5$ 时，矩形 $OAMB$ 和 $\triangle OPQ$​​ 的重叠部分面积是否能够等于 4.1？

长沙 2019 / 抛物线 $y=ax^2+6ax (a>0)$ 与 $x$ 轴交于 $A,O$ 两点，$B$ 为抛物线顶点，$D(t,0) (-3<t<0)$ ，连接 $BD$ 并延长过 $O,A,B$ 三点的 $\odot P$ 相交于点 $C$。过点 $C$ 作 $\odot P$ 的切线 $CE$ 交 $x$ 轴于点 $E$，连接 $AC,BE,BO$。

1 / 求点 $A$ 的坐标。

2 / 求证 $CE=DE$ 。

3 / 当 $a=\dfrac{\sqrt{3}}{3}, \angle CAE=\angle OBE$ 时，求 $\dfrac{1}{OD}-\dfrac{1}{OE}$ 的值。

答案

邵阳 2019 / 如图，在此给出 3 中合法的平行四边形。

1 / 由顶点式得 $y=-\dfrac{1}{3}x(x-8)$，即 $y=-\dfrac{1}{3}x^2+\dfrac{3}{8}x$。

2 / 设 $A(a,m)$，则 $B(8-a,m)$。由题，$AB=8-2a=AD=m, \therefore a=\dfrac{8-m}{2}$ 。

$\therefore A(\dfrac{8-m}{2},m)$，代入抛物线得 $m=-\dfrac{(8-m)^2}{2}+\dfrac{8(8-m)}{6}$ ，整理得

$(m+16)(m-4)=0, m=4$ 或 $m=-16$ (舍)。由此，$A(2,4),B(6,4),C(6,0),D(2,0)$ 。

3 / 显然，当 $P$ 在 $AB$ 内时无解。由两点式，$AC:y=6-x$。我们设 $P(a,4) (a=t+2)$。

计算可得 $E(a,\dfrac{8a-a^2}{3}),F(a,6-a),Q(2,a-6)$。由平行四边形的判定，

$|EF|=|AQ|\Rightarrow \dfrac{8a-a^2}{3}-(6-a)=10-a$，整理得 $(a-6)(a-8)=0,a=6$ 或 $a=8$。

所以，当 $t=4s$ 或 $t=6s$ 时，能够构成平行四边形。

长沙 2018 / 在此给出第三题的三种情况。

图 1 图 2 图 3

1 / 显然 $PQ:y=m+1-x, \angle OCD=45^\circ$ 。

2 / 由题得 $P(3,1),Q(1,3),C(4,0)$，设 $M(a,\dfrac{3}{a})$，由相似得 $\dfrac{OP}{OM}=\dfrac{OC}{OP}=\dfrac{PC}{PM}$ 。

由图，$OP=\sqrt{10},OC=4,OM=\sqrt{a^2+\dfrac{9}{a^2}}$。不难发现 $OP^2=OC\cdot OM$，代入得

$10=4\sqrt{a^2+\dfrac{9}{a^2}}$，整理得 $(4a^2-9)(a^2-4)=0,$ 又 $\because 1<a<3 \therefore a=\dfrac{3}{2}$ 或 $a=2$。

当 $a=\dfrac{3}{2}$ 时，检验失败，舍。$a=2$ 时，检验成功，$M(2,\dfrac{3}{2})$。

3 / 设 $M(a,\dfrac{5}{a})$。由题得 $P(5,1),Q(1,5),C(6,0),D(0,6)$​​​​​。

  • 当 $a<1$ 时，如图 1，$S<S_{\triangle OAM}<2.5$，不存在。

  • 当 $a>5$ 时，如图 1，$S<S_{\triangle OBM}<2.5$，不存在。

当 $1<a<5$ 时，如图 2，设 $OQ$ 交 $BM$ 与 $E$，$OP$ 交 $AM$ 于 $F$，由相似，

$E(\dfrac{1}{a},\dfrac{5}{a}),F(a,\dfrac{a}{5})$。$S=S_{OAMB}-S_{\triangle OBE}-S_{\triangle OAF}=4.1$，由此可得

$5-\dfrac{x^2}{10}-\dfrac{5}{2x^2}=4.1$，整理得 $x^4-9x^2+25=0$​​，没有实数根，不存在。

长沙 2019 / 连接并延长 $BP$ 并交 $x$ 轴于 $F$，连接 $PC$。

1 / 由解析式得零点坐标满足 $ax(x+6)=0$，所以 $A(0,0)$ (舍) 或 $A(-6,0)$ 。

2 / 由题，$\angle ECP=\angle BFD=90^\circ, \angle PBD=\angle PCB$。

$\therefore 90^\circ-\angle PBD=90^\circ - \angle PCB, \angle FDB=\angle CDE=\angle DCE, EC=ED$。

3 / 设 $E(a,0)$，由切割线定理，$CE^2=AE\cdot OE$。由图，$\angle CAO=\angle CBO=\angle OBE$。

由角平分线定理，$\dfrac{BD}{BE}=\dfrac{DO}{OE}$。由解析式，$B(3,-3\sqrt{3})$。由两点间距离公式，

$BD=\sqrt{(3+t)^2+27}, BE=\sqrt{(3+a)^2+27}, CE=DE=a-t$。

$\therefore CE^2=AE\cdot OE\Rightarrow (a-t)^2=a(a+6)\Rightarrow a=\dfrac{t^2}{6-2t}$。

$\therefore \dfrac{BD}{BE}=\dfrac{DO}{OE}\Rightarrow \dfrac{\sqrt{(3+t)^2+27}}{\sqrt{(3+a)^2+27}}=\dfrac{-t}{a}\Rightarrow a=-\dfrac{6t}{t+6}$。

$\therefore \dfrac{t^2}{6-2t}=-\dfrac{6t}{t+6}\Rightarrow t^2=-18t+36, \dfrac{1}{OD}-\dfrac{1}{OE}=\dfrac{1}{-t}-\dfrac{1}{-m}=\dfrac{3t+6}{t^2}=\dfrac{1}{6}$​。

挑战高考

在高中，我们不仅需要学习直线和圆，还有椭圆，双曲线，抛物线这类 圆锥曲线。原来的函数解析式变成了方程和轨迹方程，不过，解题的思路和方法依旧没有变化。你也不必太过于担心它的难度：

2015全国卷1 / 在平面直角坐标系 $xOy$ 中，曲线 $C:y=\dfrac{x^2}{4}$ 与直线 $y=kx+a (a>0)$ 交于点 $M,N$ 。

1 / $k=0$ 时，分别求 $C$ 在点 $M,N$ 处的切线方程。

2 / $y$ 轴上是否存在点 $P$，使得当 $k$ 变动时，总有 $\angle OPM=\angle OPN$？

1 / 由待定系数法得 $M(2\sqrt{a},a),N(-2\sqrt{a},a)$ 或 $M(-2\sqrt{2},a),N(2\sqrt{2},a)$ 。对 $C$ 求导，

$y’=\dfrac{1}{2}x$ 。当 $x=2\sqrt{a}$ 时切线斜率为 $\sqrt{a}$，$x=-2\sqrt{a}$ 时则为 $-\sqrt{a}$。由斜截式，

$y-a=\pm \sqrt{a}(x\mp 2\sqrt{a})$，整理得 $\sqrt{a}x-y-a=0$ 或 $\sqrt{a}x+y+a=0$。

2 / 设 $P(0,b)$ 符合题意，$M(x_1,y_1),N(x_2,y_2)$，直线 $PM,PN$ 斜率分别为 $k_1,k_2$。代入得

$kx+a=\dfrac{x^2}{4}$，整理得 $x^2-4kx-4a=0$。由韦达定理，$x_1+x_2=4k, x_1x_2=-4a$。

当 $b=-a$ 时，$k_1+k_2=0$。由斜率与角度的关系，此时 $\angle OPM=\angle OPN, P(0,-a)$。

2015全国卷2 / 已知椭圆 $C:9x^2+y^2=m^2 (m>0)$，直线 $l$ 不过原点 $O$ 且不平行于坐标轴，$l$ 与 $C$ 有两个交点 $A,B$，线段 $AB$ 的中点为 $M$ 。

1 / 证明：直线 $OM$ 斜率与 $l$ 的斜率的乘积为定值。

2 / 若 $l$ 过 $(\dfrac{m}{3},m)$，延长线段 $OM$ 与 $C$ 交于点 $P$，四边形 $OAPB$ 能否为平行四边形？请求出此时 $l$ 的斜率。

1 / 设 $l:y=kx+b,A(x_1,y_1),B(x_2,y_2),M(x_M,y_M)$。代入得

$9x^2+(kx+b)^2=m^2$，整理得 $(k^2+9)x^2+2kbx+b^2-m^2=0$，由韦达定理，

$x_M=\dfrac{x_1+x_2}{2}=-\dfrac{kb}{k^2+9}$，代入得 $y_M=\dfrac{9b}{k^2+9}$，

$k_{OM}=\dfrac{y_M}{x_M}=-\dfrac{9}{k}$，所以 $k_{OM}\cdot k=-9$，为定值。证明成立。

2 / 由 1 得 $OM:y=-\dfrac{9}{k}x$，设 $P(x_P,y_P)$ ，由待定系数法，$9x^2+(-\dfrac{9}{k}x)^2=m^2$。

整理得 $x^2=\dfrac{k^2m^2}{9k^2+81}, \therefore x_P=\dfrac{\pm km}{3\sqrt{k^2+9}}$。由平行四边形判定，需要 $x_P=2x_M$。

$l:b=y-kx=\dfrac{m(3-k)}{3}$，代入得 $x_M-kx+b=\dfrac{mk(k-3)}{3(k^2+9)}$。联立得

$\dfrac{\pm km}{3\sqrt{k^2+9}}=\dfrac{2mk(k-3)}{3(k^2+9)}$，解得 $k=4\pm\sqrt{7}$​。

（ PS：这个方程其实很好解 ）
